# 中文漢字語義化
簡繁粵中文翻譯中碰到的一些漢字差異，漢字語義化對照表記錄如下：

| 英文(en-US) | 中文简体(zh-CN) | 中文繁体(zh-TW) | 中文粤语(zh-HK) |
| -----	 | ------- | -----| -----| 
| user | 用户 | 使用者 | 用戶 |
| character | 字符 | 字元 | 字符 |
| string | 字符串 | 字元串、字串 | 字符串 |
| module | 模块 | 模組 | 糢塊 |
| default value | 默认值 | 預設值、缺省值 | 默認值 |

注：一個詞對應多個選項時，優先選擇表格內前一種。
